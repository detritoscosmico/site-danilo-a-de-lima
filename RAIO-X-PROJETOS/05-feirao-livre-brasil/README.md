# Projeto 05 — Feirão Livre Brasil

## Objetivo
Criar um grupo/comunidade de compra, venda e divulgação, com identidade própria e potencial de virar marketplace simples.

## Potencial
Médio/alto. Pode começar como grupo de Facebook e evoluir para site, catálogo local ou app.

## Estrutura recomendada

- `regras-grupo/`
- `descricao/`
- `capa-facebook/`
- `posts-fixos/`
- `moderacao/`
- `site-catalogo/`
- `monetizacao/`
