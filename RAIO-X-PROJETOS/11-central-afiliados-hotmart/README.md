# Projeto 11 — Central de Afiliados Hotmart

## Objetivo
Criar uma estrutura para divulgar produtos digitais próprios com materiais prontos para afiliados.

## Potencial
Alto. Ajuda a escalar vendas sem depender apenas do próprio tráfego.

## Produtos que podem entrar

- Mente de Titânio
- Você Contra Você
- Curso Mecânica Automotiva na Prática
- Dinheiro Enquanto Você Dorme
- Materiais de desenvolvimento pessoal

## Estrutura recomendada

- `produto/`
- `copys/`
- `banners/`
- `posts/`
- `stories/`
- `emails/`
- `regras-afiliados/`
- `pagina-afiliados/`
