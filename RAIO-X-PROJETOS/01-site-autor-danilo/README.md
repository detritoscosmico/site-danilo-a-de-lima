# Projeto 01 — Site do Autor Danilo A. de Lima

## Objetivo
Transformar o site atual em uma central profissional para divulgar livros, capturar leads e vender produtos digitais.

## Potencial
Alto. Já está publicado no GitHub Pages e pode virar a base oficial da marca autoral.

## Estrutura recomendada

- `index.html` — página principal
- `style.css` — visual do site
- `script.js` — interações
- `imagens/` — capas, foto do autor e banners
- `livros/` — páginas individuais dos livros
- `blog/` — artigos para SEO
- `captura/` — páginas de isca digital
