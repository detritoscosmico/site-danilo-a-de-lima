# Projeto 03 — Elegance Cosméticos

## Objetivo
Criar uma estrutura digital para vender cosméticos online e presencialmente, com catálogo, WhatsApp, Marketplace e identidade visual.

## Potencial
Alto. Produto físico com giro rápido, venda local e divulgação simples em grupos e Marketplace.

## Canais

- Facebook Marketplace
- Grupos de bazar locais
- WhatsApp Business
- Instagram @elegancecosmeticosbr
- Site/catálogo simples

## Produtos iniciais

- Malbec
- Floratta
- Lily
- Body Splash
- Nativa SPA
