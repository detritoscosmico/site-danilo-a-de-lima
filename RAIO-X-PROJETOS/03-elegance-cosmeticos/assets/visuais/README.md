# Visuais do Catálogo Elegance Cosméticos

Esta pasta guarda imagens SVG criadas para deixar o catálogo visualmente apresentável.

## Importante

As imagens desta pasta são visuais próprios/provisórios por categoria. Elas não são fotos oficiais dos produtos nem cópias do site O Boticário.

## Arquivos criados

- `perfume-masculino.svg`
- `perfume-feminino.svg`
- `combo-presente.svg`
- `maquiagem.svg`
- `cabelos.svg`
- `hidratante.svg`
- `infantil.svg`

## Quando usar fotos oficiais

Quando houver fotos autorizadas, salve os arquivos finais nesta pasta e substitua o `src` dos cards no arquivo:

`RAIO-X-PROJETOS/03-elegance-cosmeticos/catalogo-base.html`

## Regra

Primeiro deixar o catálogo vendável. Depois substituir os visuais provisórios por fotos reais autorizadas.
