# Site Oficial — Danilo A. de Lima

Site oficial de **Danilo A. de Lima**, criado para apresentar livros, projetos digitais, páginas de venda, acervo e presença autoral.

## Objetivo

Construir uma presença digital profissional para conectar livros, produtos digitais, projetos comerciais e canais de contato em um único endereço.

## Stack atual

- HTML
- CSS
- JavaScript
- Estrutura estática
- Preparado para publicação com GitHub Pages

## Como abrir no VS Code

1. Baixe ou clone o repositório.
2. Abra a pasta no VS Code.
3. Instale a extensão **Live Server**.
4. Clique com o botão direito em `index.html`.
5. Escolha `Open with Live Server`.

## Onde trocar links

No arquivo `index.html`, procure por:

```html
href="#"
```

Substitua pelo link real da Hotmart, Amazon, KDP, página de compra ou rede social.

## Onde trocar contato

Procure no projeto por e-mail, WhatsApp ou links temporários e substitua pelos canais oficiais que serão usados na divulgação.
