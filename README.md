# Site Oficial — Danilo A. de Lima / Universo Titânio

## Como abrir no VS Code

1. Crie uma pasta no computador chamada:
   site_danilo_a_de_lima

2. Coloque dentro dela estes arquivos:
   - index.html
   - style.css
   - script.js

3. Abra a pasta no VS Code.

4. Clique com o botão direito no arquivo index.html.

5. Escolha:
   Open with Live Server

Se não tiver o Live Server:
- Vá em Extensões
- Pesquise "Live Server"
- Instale
- Repita o passo 4

## Onde trocar links

No arquivo index.html, procure por:
href="#"

Substitua pelo link real da Hotmart, Amazon ou página de compra.
