# Regra de Organização

Cada projeto deve ter:

- `README.md`
- `docs/`
- `src/`
- `assets/`
- `roadmap.md`
- `CHANGELOG.md`

Projetos de site devem ter:

- `index.html`
- `style.css` ou `css/style.css`
- `script.js` ou `js/script.js`
- `imagens/`

Projetos com backend devem ter:

- `backend/`
- `frontend/`
- `database/`
- `.env.example`
- `package.json`
